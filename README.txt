The Pedalo Connector plugin is used by Pedalo Web Agency to manage and support WordPress websites.


== Description ==

The Pedalo Connector plugin enables the following functionality daily:

* Plugin status and version reporting
* WordPress core status reporting
* Theme status and version reporting 
* Plugin updates to the latest released version
* WordPress core updates to the latest version
* Theme updates to the latest version
* Site PHP version reporting

It is used by Pedalo Web Agency to manage and support WordPress websites by recording the status of core, plugins and themes and whether updates are available.  If updates are available they are applied to the site with a record kept of the updates made.


== Privacy Notice ==

This plugin does not:

== Installation ==

1. Upload entire `pedalo_connector` plugin folder to the `/wp-content/plugins/` directory OR upload the 'pedalo_connector.zip' file to the Add Plugin feature in the 'Plugins' menu in WordPress
2. Activate the plugin through the 'Plugins' menu (Plugins > Installed Plugins) in WordPress
3. Apply your site's activation key supplied by www.pedalo.co.uk in the 'API Key:' field on the plugin settings page '/wp-admin/options-general.php?page=pedalo-connector'


== Frequently Asked Questions ==

= Where do I get an activation key from? =

Keys are supplied by Pedalo Web Agency www.pedalo.co.uk as part of the support and management package. You can contact Pedalo using https://www.pedalo.co.uk/contact-us/

= Will the plugin work without an activation key supplied by Pedalo? =

The plugin without the activation key will not do anything.
